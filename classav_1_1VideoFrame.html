PixelFormat av::VideoFrame::adjustFromJpegPixelFormat ( )

Convert JPEG pixel format to the compatible non-JPEG one.

YUV Pixel formats with J in the naming (AV_PIX_FMT_YUVJxxx) mostly deprecated and some components warn about it. Allow to adjust internal pixel format into compatible one.
